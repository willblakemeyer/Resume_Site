Physics Game with Machine Learning

Description: This project simulates an environment based on a popular online game called bonk.io and it trains neural networks to play the game automatically. The neural networks and their structure were generated with TensorFlow.js, which allowed me to get some experience with this powerful machine learning library. I also implemented the feature to play against the AI in a testing mode. As of writing this, I do plan to implement this AI into the online game itself and have it play against other people.

Neural Network with Genetic Algorithm

Description: To better my understanding of neural networks, I built my own from scratch. I wrote this in JavaScript, HTML, and CSS with no libraries; I only used another person's code for a simple sorting algorithm. The end result was a semi-functional, learning, group of AIs that attempted to follow around a little character I created. There were customizable settings like the mutation rate, population, the number of successors for the next generation, etc. I had to guide them by the hand a bit in the learning process, but ultimately the project was successful and my understanding of neural networks improved.

Chess Recreation

Description: This program is essentially a computer chess game built from scratch. It draws a position, allows the player to only make legal moves, and puts the player up against a chess AI. The program understands all the rules of chess, including castling and en passant. Unfortunately, the AI is very simple, as it will just make random moves unless it can capture something. Originally I was going to implement a better chess AI, but I decided to do this as a separate project.
